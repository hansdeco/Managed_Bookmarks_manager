# Managed Bookmarks Manager

PowerShell WinForms tool to build and maintain managed browser bookmark policies for Google Chrome and Microsoft Edge.

## Features

- Visual editor for nested folders and links
- Live JSON preview for managed bookmarks
- Import and export bookmark policy JSON
- Export deployment-ready PowerShell script
- Optional direct registry read/write flows for administrators
- Autosave and recovery support

## Quick Start

1. Open PowerShell 5.1+.
2. Run `ManagedBookmarksCreator.ps1`.
3. Build your bookmark tree in the GUI.
4. Export JSON or export a deployment script.

## Requirements

- Windows (WinForms)
- PowerShell 5.1 or later
- Optional admin rights for registry write features
