# Security Policy

## Supported versions

Security fixes are applied to the latest version on the default branch.

## Reporting a vulnerability

Please do not open public issues for sensitive security reports.

Use GitHub Security Advisories to report vulnerabilities privately:
- Repository -> Security -> Advisories -> Report a vulnerability

If advisories are unavailable, open an issue with minimal details and request private follow-up.

## Response process

- Initial triage target: within 7 days
- Confirmed issues are fixed as quickly as possible
- Acknowledgment and release notes are provided after remediation
